**[Lovable.dev Optimized Prompt] – Highdeium: AI-Powered Multimedia Book Writing & Publishing Platform**

### **1. Project Overview**
**Project Name:** Highdeium
**Tagline:** *"Unleash Your Story – Any Genre, Any Format, Any Audience."*
**Description:**
Highdeium is a **next-generation book writing and publishing platform** that allows users to create, share, and monetize **multimedia-rich books** (text, images, videos, music, and covers in **up to 4K resolution**). The platform features:
- **AI-Powered Content Moderation:** An algorithm automatically detects appropriateness. If marked **18+**, content is redirected to a separate **"Mature Content" channel**.
- **Genre-Agnostic Publishing:** Supports **all genres** (fantasy, sci-fi, romance, horror, non-fiction, etc.).
- **Social Feed:** Users can follow creators, discover trending books, and engage via likes/comments.
- **Monetization:** Integrated **Stripe** for subscriptions, tips, or paywalled content.
- **Collaboration:** **GitHub** integration for version control (optional for co-authoring).
- **Backend:** **Supabase** for scalable databases, auth, and storage.

**Target Audience:**
- **Primary:** Aspiring & established **writers, authors, and content creators** (ages 13+).
- **Secondary:** Readers, artists, musicians, and indie publishers.
- **Tertiary:** Book clubs, educators, and collaborative writing groups.

**Unique Selling Points (USPs):**
✅ **Multimedia Books** – Embed videos (trailers, readings), music (soundtracks), and high-res images.
✅ **Smart Moderation** – AI flags NSFW content; users self-classify as **18+** for mature channels.
✅ **Discoverability** – Algorithm-driven **feed** with trending, recommended, and genre-specific sections.
✅ **Monetization Ready** – Stripe for payments, Supabase for secure data.

### **3. Technical Specifications**
#### **A. Integrations**
| Service       | Purpose                                                                 |
|---------------|-------------------------------------------------------------------------|
| **Stripe**    | Payments (subscriptions, tips, purchases).                             |
| **Supabase**  | Auth, databases, storage (for media), real-time updates.               |
| **GitHub**    | Version control for collaborative writing (optional).                 |
| **Cloudinary**| (Optional) Image/video optimization & CDN.                             |

#### **B. Database Schema (Key Tables)**
| Table          | Fields (Example)                                                                 |
|----------------|-------------------------------------------------------------------------------|
| **Users**      | `id, username, email, role, bio, profile_pic, stripe_customer_id`          |
| **Books**      | `id, title, author_id, genre, description, cover_image, is_mature, price`  |
| **Chapters**   | `id, book_id, title, content (rich text), media_attachments, order`         |
| **Media**      | `id, url, type (image/video/audio), resolution, size, alt_text`            |
| **Transactions** | `id, user_id, book_id, amount, type (purchase/tip/subscription), date`   |
| **Engagement** | `id, user_id, book_id, type (like/comment/bookmark), content, timestamp`   |

#### **C. Security & Compliance**
- **Age Verification:** Self-declaration for 18+ content (legal disclaimer).
- **Copyright Protection:** DMCA takedown process; watermarks for premium content.
- **Data Privacy:** GDPR/CCPA compliant; Supabase’s built-in security.
